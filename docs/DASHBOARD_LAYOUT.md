## Dashboard Sections (Top to Bottom)

1. **Financial Overview** (8 cards)
   - Primary focus: Dispensary income (today, month, total)
   - Total revenue metrics (today, month)
   - Revenue breakdown by service type

2. **Patient Care** (4 cards)
   - Patient and consultation metrics
   - Appointment tracking

3. **Departments** (4 subsections)
   - **Pharmacy**: 4 cards showing medicines, prescriptions, income
   - **Laboratory**: 3 cards showing orders, completions, revenue
   - **Radiology**: 3 cards showing orders, completions, revenue
   - **IPD**: 4 cards showing admissions, beds, occupancy

4. **Alerts** (Dynamic)
   - Only displays alerts when conditions are met
   - Color-coded by severity/type

5. **System Info** (2 cards)
   - Total users and completed payments

6. **Recent Activity** (Feed)
   - Last 10 activities from across the system
   - Color-coded by activity type
   - Relative timestamps

## Responsive Behavior

- **Mobile** (< 768px): 1 column layout
- **Tablet** (768px - 1024px): 2 column layout
- **Desktop** (> 1024px): 4 column layout (financial section)

## Data Sources

All data is pulled fresh from the database on each page load:
- `patients` table
- `consultations` table
- `payments` table
- `prescriptions` table
- `medicines` & `medicine_batches` tables
